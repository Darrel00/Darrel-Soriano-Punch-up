Q: What is the work you’re taking on?
A: My idea is to create a canvas with a brush tool for drawing to my portfolio website. This component would appear on its own page where users would be able to use their mouse to create drawings that can be saved as images. There will be 2 modes for the drawing tool, one to draw with a prompt and a timer to simulate a drawing game that makes the user feel like a designer, and one that will allow the user to draw freely. On the sidebar, each component controls the pen and eraser tool, stroke length and color. Undo and redo buttons will also be added. Both of the free draw and challenge modes allow the user to save their canvas art as a downloadable image.

Q: How do you think this is going to enhance your body of work? 
A: My portfolio has a high focus on my graphic design work, and I want this idea to be a way to promote creativity and expression of ideas, which is what I try to apply in my projects. With this tool, I can demonstrate an understanding of responsive JS events while connecting it to my personal branding.

Q: Will it be a new entry, or an interesting interaction?
A: It will be a fun, interactive component on the portfolio website that will add engagement to the website, and show some of my creativity through coding.
